# Docs Directory

This directory contains project documentation for `EventifyMTEP`.

## Contents

- `README.md` — This file, which explains the purpose of the `docs` folder.

Use this folder for documentation that is separate from application code and repository-level documentation. It is a good place for:

- onboarding and environment setup guides
- architecture notes
- design decisions
- operational or deployment instructions
